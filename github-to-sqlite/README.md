# Cloudquery 101

Install cloudquery:

``` shell
brew install cloudquery/tap/cloudquery 
```

or 

``` shell
curl -L https://github.com/cloudquery/cloudquery/releases/download/cli-v2.4.1/cloudquery_darwin_amd64 -o cloudquery
chmod a+x cloudquery
```

execute job: 

``` bash
chomod +x run.sh
bash run.sh
```
